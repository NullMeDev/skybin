Built for Security

Everything you need to monitor paste sites at scale

Pattern Detection

Automatically detect API keys, credentials, private keys, and other sensitive data with configurable regex patterns.

Real-time Scraping

Concurrent scrapers fetch from multiple paste sources simultaneously with intelligent rate limiting.

Full-text Search

SQLite FTS5-powered search across all collected pastes with instant results.

Privacy First

Automatic anonymization strips PII from scraped content. No IP logging for submissions.

Auto-purge

Configurable retention policies automatically remove old pastes. FIFO enforcement caps storage.

JSON API

Full REST API for programmatic access. Submit URLs, search pastes, retrieve statistics.

config.toml
# Server configuration [server] host = "0.0.0.0" port = 8081 # Enable paste sources [sources] pastebin = true gists = true external_url = true # Pattern detection [patterns] aws_keys = true private_keys = true credentials = true
